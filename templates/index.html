Welcome to PasswordAPI

Check your password strength

To check if your password is strong or not just go to /check/{Your password} and it should be ok

Create a password

To create your password go to /crate/[query]

  • Type ? and then the first params you want to change
  • length=30 for a password of 30 characters
  • upper or digits or special to choose the option and set to true or false
  • To add more params you just need to put before &
  • Example go to http://localhost/create/?length=20&upper=true&digits=true
